## :school_satchel: Get Started

Clone the repository
```bash
git clone https://github.com/WesleyHsieh0806/TAO-Amodal.git 
```

Setup environment
```bash
conda create --name TAO-Amodal python=3.9 -y
conda activate TAO-Amodal
bash environment_setup.sh
```
